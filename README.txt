SPARK TRANSPORTATION — IPHONE TEST APP

This package is an installable Progressive Web App (PWA).

Included rules:
- Start Day only requires beginning mileage and start time.
- End Day requires ending mileage and end time.
- Live Load and Live Unload use start/stop times and decimal hours at $17/hour.
- Wait Time uses start/stop times and decimal hours at $17/hour.
- Meetings, Surveys, and Training use start/stop times and pay ADP ÷ 10 per hour.
- PTO choices: Vacation, Sick, or Safety.
- PTO pay is calculated using ADP ÷ 10 per hour.
- Manual ADP is available until enough history exists.

IMPORTANT:
An iPhone can install this on the Home Screen only after these files are hosted on an HTTPS website.
Upload the entire folder to GitHub Pages, Netlify, Vercel, or another HTTPS host.

After it is hosted:
1. Open the website in Safari on your iPhone.
2. Tap the Share button.
3. Tap “Add to Home Screen.”
4. Tap “Add.”

The app then opens from an icon like a normal iPhone app and saves its data locally on that iPhone.
